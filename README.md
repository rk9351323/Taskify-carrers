# Taskify Careers - Job Application

A simple, multi-step job application form built with HTML, CSS, and Vanilla JavaScript.

## Features

- **Multi-step Form:** Guides the applicant through Personal Info, Experience, and Review steps.
- **Dynamic File Loading:** Header and footer components are loaded dynamically.
- **Dark Mode:** Toggle switch to switch between light and dark themes.
- **Responsive Design:** Works well across desktop and mobile devices.

## Setup & Deployment

1. Clone the repository.
2. Open `index.html` in your browser (using Live Server is recommended for dynamic component loading).
3. Deployed via GitHub Pages.

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
